minuteHourChart 分时图 #

该组件内部实现是采用 canvas 进行绘制，支持通过传参来展示特定的折线。

分时图 #


html
<minute-hour-chart 
    id="minuteHour" 
    canvasId="minute-hour" 
    lineWidth="{{1}}" 
    base_info="{{base_info}}"
    list="{{list}}" 
    space="{{40}}" 
    mb="{{56}}" 
    averageColor="#FF8300" 
/>

实时绘制分时图 #


html
<minute-hour-chart 
    id="minuteHour" 
    canvasId="minute-hour" 
    lineWidth="{{1}}" 
    base_info="{{base_info}}"
    list="{{list}}" 
    space="{{40}}" 
    mb="{{56}}" 
    averageColor="#FF8300" 
/>

主要是获取到数据后调用方法对画布进行清空然后在进行渲染。

js
// 获取组件实例
this.component = this.selectComponent('#minuteHour');
// 此方法的作用是清空画布并重新绘制
this.component.clearCanvas();
